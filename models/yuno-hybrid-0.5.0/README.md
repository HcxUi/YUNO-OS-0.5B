# YUNO-Qwen Hybrid LLM v0.5.0

This is a standalone hybrid LLM combining **Qwen (QW)** base architecture with the **YUNO Personal AI OS** system layer.

- **Base Model:** `Qwen/Qwen3-0.6B`
- **Special Tokens:** `<think>`, `</think>`, `<tool_call>`, `</tool_call>`, `<tool_response>`, `</tool_response>`
- **Language Tuning:** Hinglish Native + English
- **Format:** SafeTensors Standalone CausalLM
